# Revvel Email Organizer

## Overview
Intelligent, privacy-focused email organization tool with AI-powered processing.

## Installation
```bash
pip install revvel-email-organizer
```

## Configuration
Set environment variables:
- `OPENROUTER_API_KEY`: OpenRouter API key
- `REVVEL_MASTER_KEY`: Master encryption key

## Usage
```bash
# Process email archive
revvel-email-organizer process /path/to/archive

# Generate component token
revvel-email-organizer token email_classifier

# List available AI models
revvel-email-organizer models
```

## Features
- Secure token management
- Multi-model AI integration
- Privacy-preserving processing
- Neurodivergent-friendly design
